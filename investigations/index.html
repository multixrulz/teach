Investigations

This is where I put various investigations that can be used for teaching.

How is an investigation different from an activity? An activity is a directed task you give the students. An investigation is a “thing” to investigate, like siphons, so it’s more open ended.

This is from a teacher’s point of view, so it doesn’t include things that you would give to students. They go on learn.henley.id.au.
